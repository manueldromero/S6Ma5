La forma polar de un número complejo

Forma polar de un número complejo

Un número complejo z=a+bi también puede expresarse como rheta, que representa la forma polar: z=r(cos⁡heta+isin⁡heta) donde:

  • Módulo: r=|z|=a2+b2
  • Argumento: heta=arg⁡(z) es el ángulo que forma el vector z con el eje real positivo

La forma polar es especialmente útil para interpretar gráficamente los números complejos y operar con ellos en situaciones geométricas. La notación rheta indica un número de módulo r y argumento heta, es decir, su posición en el plano complejo.

Convertir z=1+3i a forma polar

  • Parte real: a=1, parte imaginaria: b=3
  • Módulo: r=12+(3)2=4=2
  • Argumento: heta=arctan⁡(31)=π3
  • Forma polar: z=2π3

Convertir z=−2+2i a forma polar

  • Parte real: a=−2, parte imaginaria: b=2
  • Módulo: r=(−2)2+22=8=22
  • Argumento: heta=arctan⁡(2−2)=arctan⁡(−1). Como el número está en el segundo cuadrante: heta=3π4
  • Forma polar: z=223π4

Convertir a forma binómica: z=4π6

  • cos⁡π6=32, sin⁡π6=12
  • Multiplicamos: z=4(32+i⋅12)
  • Resultado: z=23+2i
  • Forma binómica: z=23+2i

¡Ojo con el cuadrante!

Sea z=−1−3i. Calcula su forma polar paso a paso

  • Parte real: a=−1
  • Parte imaginaria: b=−3
  • Módulo: r=(−1)2+(3)2=1+3=2
  • anheta=ba=−3−1=3⇒heta=arctan⁡(3)=π3
  • Pero: El número está en el tercer cuadrante, así que: heta=π+π3=4π3
  • Forma polar: z=24π3

¿Cuál es el módulo del número complejo z=−2+2i?
Respuesta correcta: 2√2
Explicación: El módulo se calcula como (−2)2+22=4+4=8=22.

¿En qué cuadrante se encuentra el número z=−2+2i?
Respuesta correcta: Segundo cuadrante
Explicación: Parte real negativa, parte imaginaria positiva → segundo cuadrante.

¿Cuál es la forma polar del número z=1+3i?
Respuesta correcta: 2π3
Explicación: El módulo es 12+(3)2=2 y el argumento es arctan⁡(3)=π3.

¿Qué número complejo corresponde a z=4π6 en forma binómica?
Respuesta correcta: 23+2i
Explicación: cos⁡(π6)=32, sin⁡(π6)=12, entonces z=4(32+i⋅12)=23+2i.
